# FH-Charset

## 概述
浏览器插件，专用于**`网页字符编码集的修改`**，比如：UTF-8、GBK、韩文、日文、西文等等。。。

如果你在网上冲浪的过程中，发现浏览到的网页出现乱码，那这个工具一定能帮助到你！

## 版本
支持`Google-Chrome`、`Microsoft-Edge`两种浏览器
